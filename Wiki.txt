Sistema de gerenciamento escolar que integra funcionalidades que vão facilitar a relação aluno/professor. Entre estas funcionalidades está o envio de mensagens, visualização de boletins, de frequências escolares, atividades, tarefas, entre outros.

Palavras-chaves:

Sistema Academico Notas Alunos professores desempenho gerenciar manter escolar, aula, frequencia, curriculo, historico

Funcionalidades:

1 - Aluno:

-> Visualizar o Boletim;
-> Criar\Visualizar as Tarefas (Aluno);
-> Receber e Enviar Mensagens;
-> Ver Histórico Escolar;
-> Usar E-mail dedicado;
-> Ver Dados Cadastrados;

2 - Professor:

-> Criar\Visualizar as Tarefas (Aluno e Professor);
-> Inserir faltas;
-> Inserir Notas;
-> Receber e Enviar Mensagens;
-> Encerrar a Disciplina;
-> Ver Calendário de Tarefas (Tarefas);
-> Usar E-mail dedicado;
-> Ver Dados Cadastrados;
-> Ver Dados dos Alunos (Limitado);

3 - Servidor:

-> Relacionar matéria, professor, aluno, turma;
-> Ver Dados Cadastrados;
-> Criar, aluno, professor, turma e matéria;

4 - Sem Supervisão do Usuário:

-> Email;
